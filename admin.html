Settings

Admin Credentials

To change admin password:

  1. Edit admin.js line ~12
  2. Update the credentials object
  3. Change password field to your new password
  4. Reload the page

File Management

To add, delete, or modify files:

  • Videos: Upload to images/videos/large/ via Git or FTP
  • Gallery: Upload to images/large/ via Git or FTP
  • Thumbnails: Add to images/thumbnails/ (optional)

GitHub API

This admin panel uses GitHub API to list files. API is rate-limited to 60 requests/hour.

Rate Limit: 60 requests per hour (per IP)
Reset: Resets hourly
